Мой набор плагинов для Figm'ы

Те, что выделены жирным, — на мой взгляд, самые-самые. Остальные, впрочем, — тоже must-have. Названия, как правило, говорят сами за себя. Не самые очевидные, я сопроводил короткими пояснениями.

Иконки, изображения, анимация

  • AEUX — импорт файлов в AE.
  • BeatFlyer Lite — плагин для простых анимационных эффектов, расширение одноименного онлайн-сервиса.
  • Material Design Icons.
  • Blobs — генератор привлекательных клякс.
  • Blush — конструктор иллюстраций.
  • Feather Icons.
  • Figmotion.
  • Humaaans for Figma.
  • Iconify.
  • LottieFiles — для импорта Lottie-анимации, как gif’ов.
  •  Pattern Hero.
  • Streamline Icons.
  • Vectary 3D Elements.

Цвет, шрифт, стили

  • arc — текст по контуру.
  • Better Font Picker.
  • Color Luminance.
  • Easing Gradients.
  • Image Palette.
  • Palette — подбор палитры.
  • SmoothShadow.
  • Themer — создание и замена тем из стилей в библиотеке.
  • Toolabs Design System Manager.
  • Typescale.

Поиск и замена

  • Batch Styler — пакетное изменение стилей
  • Brandfetch — поиск и скачивание логотипов разных компаний
  • Component Replacer.
  • Content Buddy — поиск одинакового текста
  • Find / Focus.
  • Find and Replace Colors.
  • Font Replacer.
  • Hidden Styles Remover.
  • Instance Finder.
  • Merge it.
  • Select Layers — выбор слоев по типам: текстовые, изображения, овалы и т. д.

Проверки

  • Above The Fold.
  • Design Lint.
  • Spellchecker.
  • Stark — проверка контраста по стандартам WCAG.
  • Style Organizer.

Организация объектов и слоев

  • AutoGrid создает из элементов Autolayout-структуру — “сетку”. См. также LilGrid.
  • Clean Document — очистка от мусора.
  • Component Utilities — дублирование компонента и пр.
  • LilGrid — распределяет объекты по виртуальной сетке, в отличие от AutoGrid не создает Autolayout-структуру
  • Masonry Grid Layout Organizer.
  • Move Layers — перемещение нескольких объектов с помощью цифрового ввода.
  • Organize Layers — делает тоже, что Symbol Organizer, только для любой страницы.
  • Rename It — усложненные пакетные переименования (см. также их выражения)
  • Responsify — создает из одного фрейма несколько]() — под разные экраны
  • Reverse Layer Order.
  • Safely Delete Components — удаление неиспользуемых компонентов.
  • Sort It.

“Рыба” и данные

  • Autoflow — формы и стрелки для flow-схем
  • Avatars.
  • Charts.
  • Coda for Figma — данные из Spotify, Wikipedia, Dropbox, Gmail, Google Calendar, Jira, Shopify, Github, etc.
  • Content Reel — различная “рыба”: аватары, название компаний, адреса
  • Datavizer.
  • Lorem ipsum.
  • Mapsicle — интерактивные карты.
  • Recognize — OCR, распознавание текста в изображении.
  • Table Generator.
  • Table Paste — копирование данных из Excel для вставки в компонент таблицы.
  • Unsplash.

Разное

  • Figma Tips.
  • Pitchdeck Presentation Studio — экспорт фреймов в презентацию PowerPoint.
  • Todo.
